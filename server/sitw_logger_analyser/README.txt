DESCRIPTION

There is no special installation or configuration for the the Node Action Logger software. You may put the unzipped folder to anyway you like. There are 2 .bat files in the folder:
runLog.bat - run this to log the node actions in 24/7. The logged data is stored in SitwUsageLog\log\logActions.
runAna.bat - run this to analysis the logged data and produce a visualized output.

Configuaration

SitwData.py
UnionHost = '<UNION Server IP address>'
